# 泛化项目工程

> 跨领域脚本、插件、小项目开发

## 定位

解决泛化知识的脚本、插件开发，涉及领域广泛：
- 游戏开发（Unity/Cocos/Unreal）
- 设计美术（Adobe/Blender/Spine）
- 开发工具（Kiro/VS Code/脚本）
- AI 探索（大模型/本地部署）
- 应用开发（Mobile/Desktop）

当项目开发周期长、内容深度深时，独立出去做垂直开发工程。

| `scripts/` | 脚本工具库（3dsmax, blender, photoshop, spine, windows） |
| `research/` | 研究与探索（ai-philosophy, plugin-design, comfyui, agent, experiments） |
