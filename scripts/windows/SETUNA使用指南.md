# SETUNA 截图置顶工具使用指南

## 工具简介

SETUNA 是一款来自日本的轻量级截图置顶工具，可以将屏幕任意区域截图后固定显示在桌面最上层，非常适合需要参考图片、文档或代码片段的工作场景。

## 核心功能

- **区域截图**：截取屏幕任意矩形区域
- **置顶显示**：截图作为独立窗口始终显示在最前端
- **透明度调节**：可调整截图窗口的透明度
- **自由移动**：截图窗口可随意拖动到任意位置
- **快速关闭**：右键或快捷键快速关闭截图窗口

## 基本操作

### 启动程序
双击 `SETUNA.exe` 启动，程序会最小化到系统托盘。

### 截图操作
1. **开始截图**：左键点击托盘图标或使用快捷键
2. **选择区域**：按住鼠标左键拖动选择要截取的区域
3. **完成截图**：松开鼠标，截图自动置顶显示

### 管理截图窗口
- **移动窗口**：左键拖动截图窗口
- **调整透明度**：鼠标悬停在截图上，使用滚轮调节
- **关闭窗口**：右键点击截图窗口选择关闭
- **关闭所有**：右键托盘图标选择"关闭所有截图"

## 实际应用场景

### 1. Blender/Spine 工作流
```
场景：制作角色动画时需要参考原画设计稿
操作：
1. 打开设计稿图片
2. 使用 SETUNA 截取关键部位（头部、手部、姿势等）
3. 将截图置顶在 Blender 工作区旁边
4. 边看参考边建模/调整骨骼
```

### 2. ComfyUI 工作流调试
```
场景：调试复杂的 ComfyUI 节点配置
操作：
1. 截取正确的节点连接配置
2. 置顶显示在屏幕一侧
3. 在另一个工作流中对照配置
4. 避免频繁切换窗口
```

### 3. AI 图像生成对比
```
场景：对比不同参数生成的图像效果
操作：
1. 截取第一次生成的满意结果
2. 置顶显示
3. 继续调整参数生成新图
4. 实时对比效果差异
```

### 4. 代码参考
```
场景：编写脚本时需要参考 API 文档
操作：
1. 截取 API 文档关键部分
2. 置顶在代码编辑器旁边
3. 无需频繁切换窗口查看文档
```

### 5. 色板参考
```
场景：UI 设计或材质调色
操作：
1. 截取标准色板或参考图的颜色
2. 置顶显示
3. 使用吸管工具直接从截图取色
```

## 高级技巧

### 多截图管理
- 可以同时置顶多个截图窗口
- 建议不超过 3-4 个，避免遮挡工作区
- 按使用频率排列截图位置

### 透明度设置
- **完全不透明**：需要清晰查看细节时
- **半透明**：需要透过截图看到下层内容时
- **高透明**：仅作为位置参考时

### 快速工作流
```
1. 准备阶段：截取所有需要的参考图
2. 布局阶段：将截图排列在屏幕边缘
3. 工作阶段：专注主工作区，需要时瞥一眼参考
4. 清理阶段：完成后一键关闭所有截图
```

## 常见问题

### Q: 截图窗口无法关闭？
A: 右键点击截图窗口，选择"关闭"或"删除"

### Q: 如何设置开机自启动？
A: 将 SETUNA.exe 快捷方式放入 Windows 启动文件夹
```
路径：C:\Users\[用户名]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
```

### Q: 截图质量模糊？
A: SETUNA 截取的是屏幕显示内容，确保：
- 原图在屏幕上以 100% 缩放显示
- 显示器分辨率设置正确
- Windows 缩放比例为 100%

### Q: 快捷键冲突？
A: 右键托盘图标 → 设置 → 修改快捷键

## 推荐配置

### 针对你的工作流程
```
工作场景：Blender + ComfyUI + Spine 动画制作

推荐设置：
- 快捷键：Ctrl + Shift + S（避免与其他软件冲突）
- 默认透明度：80%（既能看清又不完全遮挡）
- 截图位置：屏幕右侧 1/4 区域
- 同时截图数：2-3 个（参考图 + 配置图）

典型工作流：
1. 打开 ComfyUI 工作流 → 截取节点配置
2. 打开参考角色图 → 截取关键姿势
3. 在 Blender 中工作，参考置顶截图
4. 完成后导出到 Spine
```

## 替代方案

如果 SETUNA 不满足需求，可以考虑：

1. **Snipaste**（推荐）
   - 功能更丰富
   - 支持简单编辑
   - 免费且持续更新

2. **PixPin**（功能最强）
   - 支持 OCR 文字识别
   - 支持录屏
   - 适合需要更多功能的用户

3. **FastStone Capture**（商业软件）
   - 专业级截图工具
   - 功能全面
   - 需要付费
