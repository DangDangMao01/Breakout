# 悬浮截图工具 - 4K DPI 感知版本

替代 SETUNA，支持 4K 高分屏。

## 功能

- ✅ 区域截图
- ✅ 悬浮显示在桌面最上层
- ✅ 鼠标拖动移动
- ✅ Ctrl + 滚轮缩放
- ✅ 右键菜单（放大/缩小/重置/关闭）
- ✅ 4K DPI 自动适配

## 安装

```bash
pip install -r requirements.txt
```

## 使用

### 方法 1：双击运行
```
启动悬浮截图.bat
```

### 方法 2：命令行
```bash
python FloatingScreenshot.py
```

## 操作说明

1. **截图**：运行后，鼠标拖动选择区域
2. **移动**：左键拖动悬浮窗口
3. **缩放**：Ctrl + 鼠标滚轮
4. **菜单**：右键打开菜单
5. **取消**：按 ESC 取消截图

## 快捷键

| 操作 | 快捷键 |
|------|--------|
| 取消截图 | ESC |
| 放大 | Ctrl + 滚轮上 |
| 缩小 | Ctrl + 滚轮下 |

## 系统要求

- Windows 10/11
- Python 3.7+
- PyQt5
