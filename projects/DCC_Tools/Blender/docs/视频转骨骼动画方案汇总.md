# 视频转骨骼动画方案汇总

> 目标：从视频自动生成骨骼动画，导入 Blender，再通过 FBX2Spine 转到 Spine

---

## 工作流程

```
视频文件 (MP4/MOV)
     ↓
AI 动捕工具（提取骨骼动画）
     ↓
FBX/BVH 文件
     ↓
Blender 导入
     ↓
调整和优化
     ↓
导出 FBX
     ↓
FBX2Spine 转换
     ↓
Spine 2D 动画
```

---

## 方案一：Mixamo（免费，推荐）

### 功能
- 在线自动绑定
- 免费动作库（2000+ 动画）
- 无需视频，直接使用预设动画

### 使用步骤

```
1. 准备 3D 模型
   - Blender 创建简单人形模型
   - 或使用现有模型

2. 上传到 Mixamo
   - 访问：https://www.mixamo.com
   - 注册免费 Adobe 账号
   - 上传模型（OBJ/FBX）

3. 自动绑定
   - Mixamo 自动识别骨骼
   - 调整关键点（如果需要）
   - 点击 "Next"

4. 选择动画
   - 浏览动画库
   - 搜索：walk, run, jump, attack 等
   - 预览动画

5. 下载 FBX
   - 选择格式：FBX for Unity
   - Skin: With Skin
   - Frames per second: 30
   - 点击 "Download"

6. Blender 导入
   - File → Import → FBX
   - 选择下载的 FBX

7. 使用 FBX2Spine
   - 按照教程转换到 Spine
```

### 优点
- ✅ 完全免费
- ✅ 2000+ 专业动画
- ✅ 自动绑定
- ✅ 质量高
- ✅ 无需视频

### 缺点
- ❌ 只支持人形角色
- ❌ 不能自定义动作（只能用预设）
- ❌ 需要上传模型

## 完整示例：Mixamo 工作流

### 步骤 1：准备模型（Blender）

```
1. 打开 Blender
2. 创建简单人形：
   - 身体：立方体拉伸
   - 头：球体
   - 手臂：圆柱体
   - 腿：圆柱体
3. 不需要绑定骨骼（Mixamo 会自动绑定）
4. 导出 FBX：
   - File → Export → FBX
   - 保存为 character.fbx
```

---

### 步骤 2：Mixamo 处理

```
1. 访问 https://www.mixamo.com
2. 注册/登录 Adobe 账号
3. 点击 "Upload Character"
4. 选择 character.fbx
5. 等待上传

自动绑定：
6. Mixamo 自动识别身体部位
7. 调整标记点（如果需要）：
   - 下巴
   - 手腕
   - 肘部
   - 膝盖
   - 脚踝
8. 点击 "Next"
9. 等待绑定完成（1-2 分钟）

选择动画：
10. 浏览动画库
11. 搜索 "walk"
12. 点击动画预览
13. 调整参数：
    - Overdrive（夸张程度）
    - Arm Space（手臂空间）
    - Character Arm-Space（角色手臂空间）
14. 满意后点击 "Download"

下载设置：
15. Format: FBX for Unity
16. Skin: With Skin
17. Frames per second: 30
18. Keyframe Reduction: none
19. 点击 "Download"
```

---

### 步骤 3：Blender 导入

```
1. 打开 Blender
2. File → Import → FBX (.fbx)
3. 选择从 Mixamo 下载的 FBX
4. 导入设置：
   - ☑ Automatic Bone Orientation
   - ☑ Import Animation
5. 点击 "Import FBX"

检查：
6. 选中骨架
7. 切换到 Animation 工作区
8. 空格键播放动画
9. 确认动画正常
```

---

### 步骤 4：调整为 2D 风格（重要！）

```
⚠️ 关键步骤：清理 Z 轴运动

1. 选中骨架
2. 切换到 Graph Editor
3. 选择所有骨骼的 Z Location 曲线
4. 删除或设为 0
5. 保留 X 和 Y 的运动
6. 调整旋转，使其适合 2D 视角

或者：
1. 在 3D 视图切换到正交视图（小键盘 1）
2. 只保留正面可见的运动
3. 删除深度运动
```

---

### 步骤 5：导出 FBX

```
1. 选中骨架
2. File → Export → FBX (.fbx)
3. 导出设置：
   - ☑ Selected Objects
   - ☑ Baked Animation
   - ☑ Key All Bones
4. 保存为 walk_cleaned.fbx
```

---

### 步骤 6：FBX2Spine

```
1. 启动 FBX2Spine
2. 导入 walk_cleaned.fbx
3. 导入 Spine character.json
4. 链接骨骼
5. 导出到 Spine
```

---

### 步骤 7：Spine 使用

```
1. Spine 导入动画
2. 播放查看
3. 微调
4. 导出到游戏
```
