# TouchDesigner 项目

**用途**: 实时视觉效果、互动装置、程序化动画

---

## 📁 文件夹结构

```
TouchDesigner/
├── projects/         # TouchDesigner 项目文件 (.toe)
├── components/       # 可复用组件 (.tox)
├── scripts/          # Python 脚本
├── assets/           # 资源文件
│   ├── images/      # 图片素材
│   ├── videos/      # 视频素材
│   └── audio/       # 音频素材
├── exports/          # 导出的视频和图片
├── presets/          # 预设和模板
└── docs/            # 文档和教程
```

## 🔧 工作流

### 实时视觉流程
```
设计效果 → 搭建网络 → 参数调整 → 实时预览 → 导出/集成
```

### 互动装置流程
```
传感器输入 → 数据处理 → 视觉反馈 → 优化性能 → 现场部署
```

### 游戏集成流程
```
TouchDesigner → OSC/UDP → Unity/Unreal
TouchDesigner → Spout/Syphon → 游戏引擎
```
