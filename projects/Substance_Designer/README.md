# Substance Designer 项目

**用途**: PBR 材质制作、程序化纹理生成

---

## 📁 文件夹结构

```
Substance_Designer/
├── materials/          # 材质文件 (.sbs)
├── textures/          # 导出的纹理
├── presets/           # 预设和模板
├── scripts/           # 自动化脚本
└── docs/             # 文档和教程
```

---

## 🎯 主要用途

### 1. PBR 材质制作
- Base Color（基础色）
- Metallic（金属度）
- Roughness（粗糙度）
- Normal（法线）
- Height（高度）
- AO（环境光遮蔽）

### 2. 程序化纹理
- 地面材质（石头、泥土、草地）
- 墙面材质（砖墙、混凝土）
- 金属材质（生锈、磨损）
- 有机材质（木头、皮革）

### 3. 游戏引擎集成
- Unity 材质导出
- Unreal Engine 材质导出
- Cocos2D 纹理优化

---

## 🔧 工作流

### 标准流程
```
设计材质 → 调整参数 → 导出纹理 → 导入引擎 → 测试效果
```

### 批量导出
```
多个材质 → 统一设置 → 批量导出 → 自动命名
```
