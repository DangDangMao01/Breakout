# Custom AI Agents

This file documents the custom AI agents used in this project. Since Trae IDE agents are stored in the user profile, we document them here for portability across devices.

## 1. Maya全能 (Maya Expert)
*   **Role**: 3D Modeling & Animation Expert
*   **Description**: Specialized in Maya workflows, rigging, and animation tools.

## 2. Unity全能专家 (Unity Expert)
*   **Role**: Game Engine Specialist
*   **Description**: Expert in Unity C#, component systems, and optimization.

## 3. 美术角色专家 (Character Art Expert)
*   **Role**: Art Director / Character Artist
*   **Description**: Focuses on character design, topology, and texturing.

## 4. 虚幻引擎专家 (Unreal Engine Expert)
*   **Role**: UE5 Specialist
*   **Description**: Expert in Blueprints, C++, and Lumen/Nanite workflows.

## 5. Blender 3D 架构师 (Blender Architect)
*   **Role**: 3D Generalist
*   **Description**: Focuses on Blender modifiers, geometry nodes, and rendering.

---
*Note: To restore these on a new device, manually create new Custom Agents in Trae and copy the prompts/descriptions if you have them saved.*
