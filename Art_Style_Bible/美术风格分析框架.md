# 游戏美术风格分析框架

> 用于系统化分析和整理游戏美术风格的标准模板

---

## 📋 使用说明

这个框架帮你：
1. **统一分析维度** - 每次分析都从相同角度入手
2. **积累知识库** - 分析结果可以复用和对比
3. **指导 AI** - 给 AI 明确的分析结构

## 📝 使用示例

### 示例：日系二次元风格

#### 1. 基础信息
- **风格名称**: 日系二次元
- **代表作品**: 原神、崩坏 3、碧蓝航线
- **目标平台**: 手游
- **目标受众**: 18-30 岁，男女均衡，亚洲市场

#### 2. 视觉特征

**色彩体系**:
- 主色调：明亮、饱和度高
- 配色方案：对比色为主（蓝+橙、紫+黄）
- 特殊用色：大量使用渐变、高光

**造型风格**:
- 头身比：6-7 头身
- 五官：大眼睛、小嘴、尖下巴
- 线条：流畅柔和，有明显描边（黑色或深色）

**材质表现**:
- 渲染：卡通渲染（Cel Shading）
- 光影：明暗分明，高光明显
- 纹理：平涂为主，局部手绘细节

#### 3. 动画特征
- 节奏：快速流畅，有弹性
- 夸张：适度夸张（头发飘动、衣服摆动）
- 特效：华丽、粒子丰富、色彩鲜艳

#### 4. 技术规范
- 角色分辨率：1024x1024
- 文件格式：PSD → PNG（透明背景）
- 骨骼数量：30-50 根

## 🎯 如何用这个框架指导 AI

### 方法 1：给 AI 提供框架
```
我要分析 [游戏名] 的美术风格，请按照以下框架回答：

1. 基础信息
2. 视觉特征
   - 色彩体系
   - 造型风格
   - 材质表现
3. 动画特征
4. 技术规范

请逐项分析，给出具体特征，不要泛泛而谈。
```

### 方法 2：让 AI 填表
```
我给你一个美术风格分析模板，请帮我分析 [游戏名]，
并填写到模板的每一项中。

[粘贴模板]
```

### 方法 3：对比分析
```
我已经分析了游戏 A 的风格（见附件），
现在请用相同的框架分析游戏 B，
并对比两者的异同。
```
