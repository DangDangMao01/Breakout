## 💡 使用技巧

### 1. 组合使用
```
第一轮：用【色彩分析】模板
第二轮：用【造型分析】模板
第三轮：用【材质分析】模板
最后：用【整体风格定位】模板总结
```

### 2. 迭代优化
```
第一次：用基础版模板快速分析
第二次：用进阶版模板深入分析
第三次：用对比分析模板找差异
```

### 3. 建立知识库
```
每次分析的结果保存为：
Art_Style_Bible/[游戏名]/
├─ 色彩分析.md
├─ 造型分析.md
├─ 材质分析.md
└─ 参考图/
```

### 4. 团队协作
```
把模板分享给团队成员：
- 美术用相同的维度分析参考
- 策划用相同的语言描述需求
- 程序用相同的标准实现效果
```

## 📝 实战示例

### 示例 1：分析原神角色

**提问**:
```
请分析原神角色"雷电将军"的【色彩体系】：
1. 主色调是什么颜色？
2. 饱和度是高/中/低？
3. 明度是亮/暗/中间调？
4. 配色方案是对比色还是邻近色？
```

**AI 回答**（结构化）:
```
1. 主色调：紫色（冷色调）
2. 饱和度：中等（不过分鲜艳，保持优雅）
3. 明度：中间调偏暗（符合角色威严的定位）
4. 配色方案：
   - 主色：深紫色（衣服）
   - 辅色：金色（装饰）
   - 点缀色：粉紫色（头发渐变）
   - 关系：紫+金为对比色，营造华贵感
```

### 示例 2：对比两个游戏

**提问**:
```
请对比分析原神和崩坏 3 的【造型风格】差异：

| 维度 | 原神 | 崩坏 3 | 差异说明 |
|------|------|--------|----------|
| 头身比 | | | |
| 五官特征 | | | |
| 线条风格 | | | |
| 细节丰富度 | | | |

请填写表格。
```

**AI 回答**（表格化）:
```
| 维度 | 原神 | 崩坏 3 | 差异说明 |
|------|------|--------|----------|
| 头身比 | 6-7 头身 | 7-8 头身 | 崩坏更修长 |
| 五官特征 | 大眼、圆脸 | 大眼、尖下巴 | 崩坏更成熟 |
| 线条风格 | 柔和流畅 | 锐利硬朗 | 崩坏更科技感 |
| 细节丰富度 | 高（服饰复杂）| 极高（机械细节）| 崩坏更精细 |
```
