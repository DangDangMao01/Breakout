# 项目独立标准

当孵化项目满足以下条件时，考虑独立为垂直开发工程：

## 必要条件（满足 3 项以上）

- [ ] 代码量 > 500 行
- [ ] 开发周期 > 2 周
- [ ] 有独立的用户/使用场景
- [ ] 需要独立版本管理
- [ ] 功能相对完整，可独立运行

## 加分条件

- [ ] 涉及多人协作
- [ ] 需要独立的 CI/CD
- [ ] 有发布计划（商店/开源）
- [ ] 文档量较大

## 独立流程

1. 在 GitHub 创建新仓库
2. 迁移代码和文档
3. 在 `archives/` 保留来源记录
4. 更新本项目的 `incubator/README.md`
