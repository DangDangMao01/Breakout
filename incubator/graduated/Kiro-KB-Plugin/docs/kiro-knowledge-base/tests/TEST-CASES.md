### 配置项
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `centralPath` | string | "" | 中央知识库路径 |
| `autoSave` | boolean | true | 自动保存开关 |
| `autoSync` | boolean | false | 自动同步模式 |
| `idleReminderMinutes` | number | 10 | 空闲提醒间隔(分钟) |
| `errorReportEnabled` | boolean | true | 错误报告功能 |
| `backlogMode` | string | "auto" | 暂存模式 |
| `autoAnalyze` | string | "manual" | 分析时机 |
| `language` | string | "zh" | 界面语言 |
| `reminderDays` | array | [7, 30] | 待办提醒天数 |
| `staleDays` | number | 90 | 过期知识阈值天数 |
| `gitAutoSync` | boolean | false | Git 自动同步 |
| `graphEdgeThreshold` | number | 5 | 知识关联图边权重阈值(1-20) |
