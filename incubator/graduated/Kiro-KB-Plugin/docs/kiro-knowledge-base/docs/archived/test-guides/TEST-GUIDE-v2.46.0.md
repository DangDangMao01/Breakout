## 📦 安装步骤

1. **卸载旧版本**（如果已安装）
   ```
   Ctrl+Shift+X → 找到 Kiro Knowledge Base → 卸载
   ```

2. **安装 v2.46.0**
   ```
   Ctrl+Shift+X → ... → 从 VSIX 安装
   选择: kiro-knowledge-base/extension/kiro-knowledge-base-2.46.0.vsix
   ```

3. **重新加载窗口**
   ```
   Ctrl+Shift+P → Reload Window
   ```

### 2️⃣ TF-IDF 语义搜索测试

#### 测试 1: 基本搜索
1. 按 `Ctrl+Alt+K` 打开快速搜索
2. 选择 "🤖 语义搜索 (TF-IDF)"
3. 输入查询: `Unity Shader 优化`
4. 检查：
   - [ ] 搜索速度 < 1秒
   - [ ] 结果按相关度排序
   - [ ] 显示相关度评分
   - [ ] 显示匹配词

#### 测试 2: 自然语言查询
查询示例：
- `如何在 VSCode 中开发插件`

检查：
- [ ] 能理解自然语言

#### 测试 4: 对比关键词搜索
