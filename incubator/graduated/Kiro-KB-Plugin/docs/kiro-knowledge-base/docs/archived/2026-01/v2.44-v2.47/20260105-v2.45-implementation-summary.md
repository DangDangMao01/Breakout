## 功能特性

### 智能筛选
- 基于标题、标签、领域匹配
- 支持中英文关键词
- 自动过滤低质量内容

### 难度分析
- 4 个难度级别：入门/进阶/高级/专家
- 基于关键词检测
- 基于代码复杂度判断

### 时间估算
- 中文: 400 字/分钟
- 英文: 200 词/分钟
- 代码: 2 分钟/块
- 最少 5 分钟

### 依赖分析
- 自动构建前置知识
- 按难度递进排序
- 确保学习路径合理

### 关键点提取
- 从二级标题提取
- 从列表项提取
- 最多 3 个关键点/步骤

## 技术亮点

### 1. 智能关键词匹配
```typescript
// 支持中英文关键词
const beginnerKeywords = [
    '入门', '基础', '简介', '介绍', '初学',
    'introduction', 'basics', 'getting started', 'beginner'
];
```

### 2. 多维度难度评分
```typescript
// 标题关键词权重更高
if (beginnerKeywords.some(kw => title.includes(kw))) score -= 2;

// 代码复杂度
if (totalCodeLines > 100) score += 2;
else if (totalCodeLines > 50) score += 1;
```

### 3. 精确时间估算
```typescript
const chineseMinutes = chineseChars / 400;
const englishMinutes = englishWords / 200;
const codeMinutes = codeBlocks * 2;
const totalMinutes = chineseMinutes + englishMinutes + codeMinutes;
```

### 4. 智能依赖构建
```typescript
// 如果当前步骤难度更高，则依赖前一步
if (difficultyOrder[currentStep.difficulty] > difficultyOrder[previousStep.difficulty]) {
    currentStep.prerequisites.push(previousStep.title);
}
```
