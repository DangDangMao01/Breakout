## 🎉 新增功能

### 1. 学习路径生成器 ✅

**命令**: `Kiro KB: 生成学习路径`  
**快捷键**: 无  
**图标**: 🎓 (mortar-board)

**功能描述**:
根据学习主题自动生成结构化的学习路径，包括：
- 智能筛选相关知识
- 自动难度分析（入门/进阶/高级/专家）
- 精确时间估算
- 前置知识依赖分析
- 关键点提取
- Markdown 格式报告

**使用场景**:
- 学习新技术栈
- 系统复习知识
- 项目前的知识准备
- 面试准备

**示例**:
```
输入主题: Unity Shader
选择难度: 全部

生成结果:
📚 Unity Shader 开发学习路径
- 8 个步骤
- 预计 4.5 小时
- 难度: 入门到高级
```

---

### 2. 健康度仪表盘 ✅

**命令**: `Kiro KB: 健康度仪表盘`  
**快捷键**: 无  
**图标**: ❤️

**功能描述**:
全面评估知识库健康状况，包括：
- 7 个关键指标监控
- 活跃度统计
- 质量分布分析
- 领域平衡分析
- 智能告警系统
- 综合评分（0-100）

**7 个关键指标**:
1. 知识总量（目标 ≥50 篇）
2. 月活跃度（目标 ≥10 篇/月）
3. 高质量占比（A/B 级占比）
4. 平均质量分（目标 ≥7 分）
5. 领域均衡度（避免过度集中）
6. 内容新鲜度（180 天内更新）
7. 标签丰富度（平均 ≥3 个/篇）

**健康度等级**:
- 🏆 A 级: ≥85 分（优秀）
- ✅ B 级: ≥70 分（良好）
- 📝 C 级: ≥55 分（及格）
- ⚠️ D 级: ≥40 分（需改进）
- ❌ F 级: <40 分（较差）

**示例报告**:
```markdown
# 知识库健康度报告

## 总体评估
- 综合评分: 85/100 (A级) 🏆
- 评估时间: 2026-01-05

## 关键指标
| 指标 | 数值 | 状态 | 说明 |
|------|------|------|------|
| 知识总量 | 92篇 | 🟢 | 知识积累充足 |
| 月活跃度 | 12篇/月 | 🟢 | 知识更新活跃 |
| 高质量占比 | 68% | 🟢 | A/B级知识占比 |
| 平均质量分 | 7.5分 | 🟢 | 整体质量优秀 |
| 领域均衡度 | 72% | 🟢 | 主要领域: kiro (35%) |
| 内容新鲜度 | 85% | 🟢 | 7篇超过180天未更新 |
| 标签丰富度 | 3.2个/篇 | 🟢 | 共 45 个不同标签 |
```

## 🎯 使用指南

### 学习路径生成器

**步骤 1**: 打开命令面板
```
Ctrl+Shift+P (Windows/Linux)
Cmd+Shift+P (Mac)
```

**步骤 2**: 输入命令
```
Kiro KB: 生成学习路径
```

**步骤 3**: 输入主题
```
例如: Unity Shader, React Hooks, Docker
```

**步骤 4**: 选择难度
```
🌱 入门级
🌿 进阶级
🌳 高级
🎯 全部
```

**步骤 5**: 查看结果
- 在新标签页打开 Markdown 报告
- 包含学习步骤、时间估算、关键点

---

### 健康度仪表盘

**步骤 1**: 打开命令面板
```
Ctrl+Shift+P

```

**步骤 3**: 等待分析
- 显示进度通知

- 计算各项指标

**步骤 4**: 查看报告
- Markdown 格式报告
