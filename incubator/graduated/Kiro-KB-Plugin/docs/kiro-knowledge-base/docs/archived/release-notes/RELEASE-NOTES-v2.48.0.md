## ✨ 新功能

### 1️⃣ 🔍 智能搜索建议

基于历史记录的智能搜索建议，让你的搜索更快更准！

**核心特性**:
- 🎯 基于历史记录的智能建议
- 📝 支持前缀匹配、包含匹配、分词匹配
- 🌐 中英文混合支持
- ⚡ 实时建议（< 10ms）
- 🔥 无输入时显示热门搜索

**工作原理**:
```
用户输入 "Unity" 
  ↓
系统分析历史记录
  ↓
建议:
  1. Unity Shader 优化 (精确前缀)
  2. Unity 性能分析 (精确前缀)
  3. 游戏引擎 Unity 教程 (包含匹配)
```

**匹配策略**:
1. **精确前缀匹配**: 查询是历史记录的开头
2. **包含匹配**: 查询包含在历史记录中
3. **分词匹配**: 中英文分词后的模糊匹配

---

### 2️⃣ 📊 搜索统计报告

全面的搜索统计，了解你的搜索习惯！

**命令**: `Kiro KB: 搜索统计`

**统计内容**:
- 📈 总体统计（总次数、模式分布、平均结果数）
- 🔥 热门搜索（Top 10）
- 🕐 最近搜索（最近 10 次）
- 💡 使用建议

**报告示例**:
```markdown
# 📊 搜索统计报告

## 📈 总体统计

| 指标 | 数值 |
|------|------|
| 总搜索次数 | 45 |
| 关键词搜索 | 30 (67%) |
| 语义搜索 | 15 (33%) |
| 平均结果数 | 8 |

## 🔥 热门搜索

1. Unity Shader 优化
2. VSCode 插件开发
3. React Hooks 最佳实践
4. TypeScript 类型系统
5. Git 分支管理

## 🕐 最近搜索

1. 🤖 数据库设计原则 - 2 小时前 (12 个结果)
2. 🔍 VSCode 快捷键 - 昨天 (5 个结果)

## 💡 使用建议

- 🤖 尝试使用语义搜索，可能会找到更相关的结果
- ⭐ 你最常搜索 "Unity Shader 优化"，考虑添加到收藏夹

### 3️⃣ 🔗 智能链接建议（已有功能增强）

Wiki 链接补全功能已在之前版本实现，本版本确保其正常工作。

**使用方法**:
```
1. 在 Markdown 文件中输入 [[
2. 自动触发补全
3. 输入文件名的一部分
4. 选择文件
5. 自动补全为 [[文件名]]
```

**特性**:
- ⚡ 实时补全
- 🔍 模糊搜索
- 📝 显示文件路径
- 📄 显示文件标题
- 🎯 智能过滤

1. 下载 `kiro-knowledge-base-2.48.0.vsix`
2. 打开 VS Code / Kiro
3. 按 `Ctrl+Shift+X` 打开扩展面板
4. 点击 `...` → `从 VSIX 安装`
5. 选择下载的文件
6. 重新加载窗口

```
操作:

  2. 选择搜索模式
  3. 开始输入
