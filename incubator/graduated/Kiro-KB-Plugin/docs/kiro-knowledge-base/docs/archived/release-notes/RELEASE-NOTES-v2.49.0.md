## ✨ 新功能

### 1️⃣ 📁☁️ 双区域面板布局

终于来了！侧边栏面板现在明确区分"当前项目"和"中央知识库"两个功能区域！

**核心特性**:
- 📁 **当前项目区域** - 专注于项目级知识管理
  - 💬 对话整理（显示待整理数量）
  - 🔍 项目内搜索（仅搜索当前项目）
  - 📂 文档浏览（按分类浏览本地知识文件）
  - 🔧 整理知识库（智能检测问题）
  
- ☁️ **中央知识库区域** - 全局知识管理
  - 🔍 全局搜索（搜索所有知识库）
  - 💡 搜索建议（基于历史的智能建议）
  - 📊 搜索统计（查看搜索习惯）
  - 🔥 热门知识（Top 10 最常访问）
  - 📁 按项目浏览（按项目类型分组）

**使用方法**:
```
1. 打开 Knowledge Base 侧边栏
2. 看到两个区域：📁 当前项目 和 ☁️ 中央知识库
3. 点击展开/折叠区域
4. 状态会自动保存，下次打开保持原样
```

**界面示例**:
```
📚 Knowledge Base
━━━━━━━━━━━━━━━━━━━━
📁 当前项目: MyProject
  ├─ 💬 对话整理 (3 待整理)
  ├─ 🔍 项目内搜索
  ├─ 📂 文档浏览 (15 个文件)
  └─ 🔧 整理知识库
━━━━━━━━━━━━━━━━━━━━
☁️ 中央知识库 (92)
  ├─ 🔍 全局搜索
  ├─ 💡 搜索建议
  ├─ 📊 搜索统计
  ├─ 🔥 热门知识 (Top 10)
  └─ 📁 按项目浏览
```

### 2️⃣ 🔥 热门知识追踪

自动追踪你最常访问的知识，快速找到常用内容！

**核心特性**:
- 📊 自动记录文件访问次数
- 🏆 显示 Top 10 最常访问的知识
- ⏰ 记录首次和最后访问时间
- 🔄 自动清理不存在的文件
- 💾 跨会话持久化存储（最多 1000 条记录）

**使用方法**:
```
1. 正常使用插件，打开知识文件
2. 系统自动记录访问次数
3. 在 ☁️ 中央知识库 → 🔥 热门知识 查看
4. 点击即可快速打开常用知识
```

**显示格式**:
```
🔥 热门知识
  ├─ ✅ Unity Shader 优化 (访问 50 次)
  ├─ 📝 VSCode 插件开发指南 (访问 35 次)
  ├─ 💬 React Hooks 最佳实践 (访问 28 次)
  └─ ...
```

### 3️⃣ 🔍 智能搜索范围

搜索功能现在支持明确的范围选择！

**搜索模式**:
- **项目内搜索** (`scope: 'local'`)
  - 仅搜索当前项目的 `knowledge-base/` 目录
  - 适合查找项目特定的知识
  - 从"📁 当前项目"区域触发
  
- **全局搜索** (`scope: 'all'`)
  - 搜索本地 + 中央知识库
  - 适合查找跨项目的通用知识
  - 从"☁️ 中央知识库"区域触发

**搜索结果标注**:
- 📁 图标 - 来自当前项目
- ☁️ 图标 - 来自中央知识库
- 显示相关度评分
- 支持关键词和语义两种搜索模式

---

### 4️⃣ 💡 搜索建议（预留）

基于搜索历史的智能建议功能（v2.48.0 已实现，v2.49.0 集成到面板）

**功能**:
- 显示最近 5 条搜索建议
- 支持前缀匹配、包含匹配、分词匹配
- 无输入时显示热门搜索
- 中英文混合支持

---

### 5️⃣ ⚙️ 状态持久化

面板状态自动保存，下次打开保持原样！

**持久化内容**:
- 📁 当前项目区域展开/折叠状态
- ☁️ 中央知识库区域展开/折叠状态
- 使用 `globalState` 跨会话保存

**配置选项**:
- `kiro-kb.defaultExpandedSection` - 默认展开的区域
  - `'current'` - 默认展开当前项目（推荐）
  - `'central'` - 默认展开中央知识库
  - `'both'` - 两个都展开
- `kiro-kb.useDualPanelLayout` - 是否使用双区域布局（默认 true）

## 🎯 使用场景

### 场景 1: 项目开发

**需求**: 在项目中快速查找项目相关知识

**操作**:
```
1. 展开 📁 当前项目
2. 点击 🔍 项目内搜索
3. 输入关键词
4. 只看到项目相关结果
```

**效果**: 避免被其他项目的知识干扰

---

### 场景 2: 跨项目学习

**需求**: 查找某个技术栈的所有知识

**操作**:
```
1. 展开 ☁️ 中央知识库
2. 点击 🔍 全局搜索
3. 输入技术栈名称（如 "React"）
4. 看到所有项目中的相关知识
```

**效果**: 系统化学习，知识复用

---

### 场景 3: 快速访问常用知识

**需求**: 经常查看某些知识文档

**操作**:
```
1. 展开 ☁️ 中央知识库
2. 点击 🔥 热门知识
3. 看到最常访问的 Top 10
4. 点击即可打开
```

**效果**: 节省 80% 的查找时间
