## ✨ 新功能

### 📚 学习路径生成器

根据学习主题自动生成结构化的学习路径，帮助你系统化学习！

**核心特性**:
- 🎯 智能筛选相关知识（基于标题、标签、领域）
- 📊 自动难度分析（入门 🌱 / 进阶 🌿 / 高级 🌳 / 专家 🎯）
- ⏱️ 精确时间估算（中文 400 字/分钟，英文 200 词/分钟）
- 🔗 前置知识依赖分析
- 💡 关键点自动提取
- 📄 Markdown 格式报告

**使用方法**:
```
1. 打开命令面板 (Ctrl+Shift+P / Cmd+Shift+P)
2. 输入 "Kiro KB: 生成学习路径"
3. 输入学习主题（例如: Unity Shader, React Hooks, Docker）
4. 选择目标难度（入门级/进阶级/高级/全部）
5. 查看生成的学习路径报告
```

**示例输出**:
```markdown
# Unity Shader 开发 学习路径

## 路径概览
- 📚 主题: Unity Shader 开发
- 🎯 难度: 综合
- 📝 总步骤: 8 步
- ⏱️ 预计时间: 4 小时 30 分钟

## 学习步骤

### 第 1 步: Shader 基础概念 🌱
- 难度: 入门
- 时间: 20 分钟
- 关键点: 顶点着色器、片元着色器、渲染管线
- 前置知识: 无

### 第 2 步: 第一个 Shader 程序 🌱
- 难度: 入门
- 时间: 30 分钟
- 关键点: ShaderLab 语法、Properties、SubShader
- 前置知识: Shader 基础概念

...
```

**适用场景**:
- 🎓 学习新技术栈
- 📖 系统复习知识
- 🚀 项目前的知识准备
- 💼 面试准备

### 📊 健康度仪表盘

全面评估知识库健康状况，提供改进建议！

**7 个关键指标**:
1. 📚 **知识总量** - 目标 ≥50 篇
2. 📈 **月活跃度** - 目标 ≥10 篇/月
3. ⭐ **高质量占比** - A/B 级知识占比
4. 📊 **平均质量分** - 目标 ≥7 分
5. ⚖️ **领域均衡度** - 避免过度集中
6. 🔄 **内容新鲜度** - 180 天内更新
7. 🏷️ **标签丰富度** - 平均 ≥3 个/篇

**健康度等级**:
- 🏆 **A 级** (≥85 分): 优秀
- ✅ **B 级** (≥70 分): 良好
- 📝 **C 级** (≥55 分): 及格
- ⚠️ **D 级** (≥40 分): 需改进
- ❌ **F 级** (<40 分): 较差

**使用方法**:
```
1. 打开命令面板 (Ctrl+Shift+P / Cmd+Shift+P)
2. 输入 "Kiro KB: 健康度仪表盘"
3. 等待分析完成（通常 2-3 秒）
4. 查看健康度报告和改进建议
```

**示例输出**:

### 核心算法

**难度判定算法**:
```typescript
// 关键词检测 + 代码复杂度
score = 0
if (标题包含"入门"关键词) score -= 2
if (标题包含"高级"关键词) score += 2
if (代码行数 > 100) score += 2

if (score <= -1) return '入门'
if (score <= 1) return '进阶'
if (score <= 3) return '高级'
return '专家'
```

**时间估算算法**:
```typescript
// 中文: 400 字/分钟
// 英文: 200 词/分钟
// 代码: 2 分钟/块

totalMinutes = 
    chineseChars / 400 + 
    englishWords / 200 + 
    codeBlocks * 2

estimatedMinutes = Math.max(5, Math.ceil(totalMinutes))
```

**健康度评分算法**:
```typescript
// 基础分 100
score = 100

// 过期占比扣分（每10%扣5分，最多30分）
score -= Math.min(30, Math.floor(staleRatio * 10) * 5)

// 待办积压扣分（每5个扣5分，最多20分）
score -= Math.min(20, Math.floor(backlogCount / 5) * 5)

// 本周无更新扣10分
if (thisWeekUpdates === 0) score -= 10

// 知识库太小扣10分
if (totalFiles < 10) score -= 10

return Math.max(0, score)
```
