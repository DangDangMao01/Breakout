## 📝 发布说明模板

```markdown
# Kiro Knowledge Base v2.45.0

## 🎉 新功能

### 学习路径生成器
根据学习主题自动生成结构化的学习路径，帮助你系统化学习。

**特性**:
- 智能筛选相关知识
- 自动难度分析（入门/进阶/高级/专家）
- 精确时间估算
- 前置知识依赖分析
- 关键点提取

**使用方法**:
1. 打开命令面板 (Ctrl+Shift+P)
2. 输入 "Kiro KB: 生成学习路径"
3. 输入学习主题
4. 选择目标难度
5. 查看生成的学习路径

### 健康度仪表盘
全面评估知识库健康状况，提供改进建议。

**特性**:
- 7 个关键指标监控
- 活跃度统计
- 质量分布分析
- 领域平衡分析
- 智能告警系统
- 综合评分（0-100）

**使用方法**:
1. 打开命令面板 (Ctrl+Shift+P)
2. 输入 "Kiro KB: 健康度仪表盘"
3. 查看健康度报告

## 🔧 改进

- 优化了侧边栏性能
- 改进了知识库扫描速度
- 完善了国际化支持

## 📦 安装

1. 下载 `kiro-knowledge-base-2.45.0.vsix`
2. 在 VS Code 中按 Ctrl+Shift+X 打开扩展面板
3. 点击右上角 `...` → `从 VSIX 安装`
4. 选择下载的文件
5. 重新加载窗口

## 🐛 已知问题

- 学习路径难度判断可能不够精确
- 大型知识库（>500 篇）性能待优化
```
