## ✨ 新功能

### 1️⃣ 🕐 搜索历史记录

终于来了！搜索历史让你的搜索更高效！

**核心特性**:
- 📜 自动记录最近 50 条搜索
- ⚡ 快速重复搜索
- 🔍 区分关键词和语义搜索
- ⏰ 显示搜索时间
- 📊 显示结果数量
- 🗑️ 一键清空历史

**使用方法**:
```
1. 按 Ctrl+Alt+K 打开快速搜索
2. 顶部显示最近 5 条搜索历史
3. 点击历史记录直接重复搜索
4. 或选择新的搜索模式
```

**界面示例**:
```
🔍 快速搜索
━━━━━━━━━━━━━━━━━━━━
📜 搜索历史:
  🕐 Unity Shader 优化
      🤖 语义 | 2 小时前 | 7 个结果
  🕐 VSCode 插件开发
      🔍 关键词 | 昨天 | 12 个结果
━━━━━━━━━━━━━━━━━━━━
🔍 关键词搜索
🤖 语义搜索 (TF-IDF)
━━━━━━━━━━━━━━━━━━━━
🗑️ 清空搜索历史
```

### 2️⃣ 📊 知识缺口分析（快捷键）

知识缺口分析功能已存在，现在添加了快捷键！

**快捷键**: `Ctrl+Alt+G`

**功能**:
- 🔍 扫描知识库，识别缺失主题
- 📈 生成覆盖率报告
- 🎯 按优先级排序缺口
- 💡 提供学习建议

**使用方法**:
```
1. 按 Ctrl+Alt+G 启动分析
2. 等待扫描完成（通常 < 2秒）
3. 查看 Markdown 格式的报告
4. 根据建议补充知识
```

**报告示例**:
```markdown
# 📊 知识库缺口分析报告

**知识总数**: 92
**整体覆盖率**: 65%

██████░░░░ 65%

## 📁 领域覆盖情况

| 领域 | 知识数 | 覆盖率 | 状态 |
|------|--------|--------|------|
| unity | 15 | ███░░ 60% | 🟡 |
| kiro | 25 | ████░ 80% | ✅ |
| web | 8 | ██░░░ 40% | 🔴 |

## 🕳️ 知识缺口

### 🔴 高优先级 (5)
- **unity/shader**: Shader基础
- **web/frontend**: React Hooks
- **database/sql**: 查询优化

### 🟡 中优先级 (12)
...

## 💡 建议
🔴 有 5 个高优先级知识缺口需要补充
   - unity/shader: Shader基础
   - web/frontend: React Hooks
   - database/sql: 查询优化
```
