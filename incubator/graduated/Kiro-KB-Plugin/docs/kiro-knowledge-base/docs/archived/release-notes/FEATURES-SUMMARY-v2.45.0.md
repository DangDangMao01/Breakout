## 🐛 已知问题

### 轻微问题

1. **学习路径难度判断**
   - 基于关键词，可能不够精确
   - 建议：在知识文件中明确标注难度

2. **大型知识库性能**
   - 超过 500 篇文档时，分析速度较慢
   - 计划：v2.46.0 优化

3. **通知点击无响应**
   - 索引生成通知无法点击
   - 建议：直接打开 INDEX.md 文件
