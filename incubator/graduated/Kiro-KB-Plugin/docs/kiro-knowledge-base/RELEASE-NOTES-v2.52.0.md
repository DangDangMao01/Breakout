# Kiro Knowledge Base v2.52.0

## 📋 版本说明

v2.52.0 专注于提升用户体验，让插件更易用、更智能、更友好。

**核心改进**：
- ✅ 首次使用引导 - 60秒完成设置
- ✅ 自动化配置 - 自动检测和修复
- ✅ 帮助系统 - 完善的帮助文档
- ✅ 智能提示 - 关键时刻的贴心提示

## ✨ 新功能

### 1. 首次使用引导 🎉

**启动时自动验证**
- 自动检测知识库配置
- 路径无效时智能提示
- 跨设备使用无障碍

**3步完成设置**
```
Step 1: 欢迎界面
Step 2: 选择/创建知识库位置
Step 3: 自动初始化完整结构
```

**自动检测知识库**
- 检查环境变量 `KIRO_KB_PATH`
- 检查常见位置（`~/Kiro-Central-KB`、`~/Documents/Kiro-KB` 等）
- 找到后自动配置，无需手动设置

### 2. 自动化配置 🤖

**一键初始化**
- 自动创建目录结构（solutions/notes/discussions/backlog）
- 自动生成 README.md 和 INDEX.md
- 自动生成 .gitignore

**自动修复**
- 检测缺失的目录并自动补全
- 检测缺失的文件并自动生成
- 保证知识库结构完整

**智能重置**
- 新增命令：`Kiro KB: 重置插件配置`
- 清除所有配置和缓存
- 不删除知识库文件（安全）
- 重新触发首次引导

### 3. 帮助系统 ❓

**帮助菜单**
- 📖 快速开始 - 3分钟上手指南
- 💡 常见问题 - 7个常见问题解答
- 🎬 视频教程 - 观看视频教程
- 🐛 报告问题 - 反馈 Bug 或建议
- 📚 完整文档 - 查看完整文档

**快速开始指南**
- 1️⃣ 保存知识（30秒）
- 2️⃣ 搜索知识（30秒）
- 3️⃣ 浏览知识（30秒）
- 💡 进阶技巧

**常见问题 FAQ**
- Q1: 跨设备使用时路径无效怎么办？
- Q2: 如何备份知识库？
- Q3: 支持哪些 IDE？
- Q4: 如何导出知识库？
- Q5: 知识库文件夹可以改名吗？
- Q6: 如何删除不需要的知识？
- Q7: 插件占用空间大吗？

**智能提示系统**
- 首次保存时显示提示
- 首次搜索时显示提示
- 跨设备使用时显示提示
- 支持"不再提示"选项

### 4. 优化的错误提示 ⚠️

**清晰的错误信息**
- 说明发生了什么
- 说明为什么发生
- 提供解决方案按钮

**示例**：
```
❌ 保存失败：知识库路径不存在

可能原因：
• 路径配置错误
• 磁盘空间不足
• 没有写入权限

[重新设置路径] [检查磁盘空间] [查看帮助]
```

## 📦 安装

### 从 VSIX 安装

```bash
# 方法1: 命令面板
Ctrl+Shift+P → Extensions: Install from VSIX
选择 kiro-knowledge-base-2.52.0.vsix

# 方法2: 命令行
code --install-extension kiro-knowledge-base-2.52.0.vsix
```

### 首次使用

1. 安装插件后重新加载窗口
2. 按照首次设置向导操作（3步完成）
3. 开始使用！

## 🔄 从旧版本升级

### 从 v2.51.0 升级

1. 卸载旧版本（可选，直接安装会自动覆盖）
2. 安装 v2.52.0
3. 重新加载窗口
4. 配置自动保留，无需重新设置

### 从更早版本升级

1. 建议先备份知识库文件夹
2. 卸载旧版本
3. 安装 v2.52.0
4. 按照首次设置向导重新配置

## 🎯 使用指南

### 快速开始

**保存知识**
```
在 Kiro 对话后 → 点击"保存对话" → 选择分类 → 完成
快捷键：Ctrl+Alt+Q 快速保存想法
```

**搜索知识**
```
侧边栏点击 🔍 搜索 → 输入关键词 → 点击结果打开
快捷键：Ctrl+Alt+F 全局搜索
```

**浏览知识**
```
侧边栏 📚 浏览 → 选择分类 → 查看文档列表
分类：✅ 问题解决 / 💬 技术探讨 / 📒 学习笔记
```

### 进阶技巧

- 使用标签快速筛选：`#unity #shader`
- 查看最近更新：侧边栏 🕐 最近
- 插入链接到代码：右键文件 → 插入链接
- 跨设备同步：使用 Git 或云盘同步知识库文件夹

### 遇到问题？

点击侧边栏 **❓ 帮助** → **💡 常见问题**

或运行命令：`Kiro KB: 帮助`

## 🐛 已知问题

### 无重大问题

v2.52.0 经过充分测试，目前无已知的重大问题。

### 注意事项

1. **跨设备使用**：插件会自动检测路径变化并提示重新设置，支持环境变量配置实现零配置跨设备使用（详见 [跨设备解决方案](./docs/v2.52.0-cross-device-solution.md)）
2. **大量文件**：超过1000个文件时，侧边栏刷新可能较慢
3. **主题兼容**：帮助文档在某些主题下样式可能不一致
