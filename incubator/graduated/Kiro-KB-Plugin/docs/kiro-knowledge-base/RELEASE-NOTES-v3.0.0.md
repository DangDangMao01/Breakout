# Kiro KB v3.0.0 Release Notes

> 2026-01-13 发布
> 
> **重大更新**：模块化重构，所有功能通过侧边栏访问

## 🎯 核心变化

### 架构重构
- ✅ **模块化设计**：extension.ts 从 11,308 行精简到 200 行
- ✅ **统一侧边栏**：所有功能通过侧边栏访问，不再依赖命令面板
- ✅ **清晰职责**：代码按功能模块组织（commands/, core/, features/, utils/, panels/）

### 用户体验改进
- ✅ **侧边栏中心化**：所有功能一目了然
- ✅ **减少命令**：从 40+ 个命令精简到 20 个
- ✅ **快捷键优化**：保留最常用的 2 个快捷键
  - `Ctrl+Alt+K` - 搜索知识库
  - `Ctrl+Alt+Q` - 快速保存

## 🔧 技术改进

### 代码质量
- **extension.ts**: 11,308 行 → 200 行（减少 98%）
- **模块化**: 每个模块 100-200 行，职责清晰
- **可维护性**: AI 可以完整读取每个模块

### 目录结构
```
extension/src/
├── extension.ts          (主入口，200 行)
├── panels/               (侧边栏面板)
│   └── knowledgePanel.ts (400 行)
├── commands/             (命令处理)
├── core/                 (核心功能)
├── features/             (辅助功能)
└── utils/                (工具函数)
```
