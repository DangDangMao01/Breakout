---

## ✨ 新功能

### 1. 文档浏览重构

**原功能**：`🔗 插入链接` - 功能定位模糊，使用场景不明确

**新功能**：`📚 知识库文档` - 清晰的文档浏览器

| 改进项 | 之前 | 之后 |
|--------|------|------|
| 名称 | 插入链接 | 知识库文档 |
| 分组 | 无分组，混乱 | 按分类分组 |
| 重复 | 有重复项 | 去重显示 |
| 点击行为 | 插入链接 | 打开文件 |
| 时间显示 | 无 | 相对时间 |

### 2. 按分类分组显示

```
📚 知识库文档 (21)
├── ✅ 问题解决 (5)
│   ├── Bug 修复记录
│   └── ...
├── 💬 技术探讨 (10)
│   ├── 架构设计讨论
│   └── ...
└── 📒 学习笔记 (6)
    ├── TypeScript 学习
    └── ...
```

### 3. 智能去重

- 同一文件只显示一次
- 按文件名去重
- 优先显示最新修改的版本

### 4. 相对时间显示

- `刚刚` - 1分钟内
- `5分钟前` - 1小时内
- `3小时前` - 24小时内
- `2天前` - 7天内
- `2026/1/9` - 超过7天

### 5. 右键菜单支持

点击文件 → 打开文件  
右键文件 → 插入链接（保留原功能）

## 🐛 已知问题

### Ollama AI 集成暂不可用

由于 Kiro IDE 对网络请求的限制，Ollama 本地 AI 功能暂时无法使用。

**影响**：
- 无法使用 AI 智能分类
- 无法使用 AI 内容摘要
- 无法使用语义搜索

**替代方案**：
- 使用关键词搜索
- 手动分类文档
- 等待后续版本支持
