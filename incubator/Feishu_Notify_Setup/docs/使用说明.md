# 美术资源自动通知系统使用说明

## 📌 这是什么？

一个自动化工具，当美术提交资源到 Git 后，自动通知对应项目的程序员。

## ✨ 解决了什么问题？

**以前**：美术提交后需要手动截图 + 飞书通知程序员  
**现在**：提交后自动发送飞书通知，无需手动操作

## 🚀 如何使用？

### 对于美术人员
1. 像往常一样提交资源到 Git
2. 系统会自动通知对应的程序员
3. 无需任何额外操作！

### 对于程序员
1. 收到飞书通知后，拉取最新资源即可
2. 通知内容包括：项目名、提交人、提交信息

## 🔧 如何修改负责人？

1. 编辑 `Feishu_Notify_Setup/owners.json` 文件
2. 修改对应项目的邮箱地址
3. 提交到 Git
4. 复制文件到 `C:\GitLab-Runner\owners.json`（或联系管理员）

## ❓ 常见问题

**Q: 我没收到通知怎么办？**  
A: 检查你的飞书邮箱是否在 owners.json 中配置正确

**Q: 我不想收到某个项目的通知？**  
A: 在 owners.json 中将该项目的邮箱改为其他人

**Q: 通知太频繁怎么办？**  
A: 可以调整触发条件，只监控特定文件夹
