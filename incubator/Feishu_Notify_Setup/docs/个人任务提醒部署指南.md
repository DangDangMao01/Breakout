## ⏰ 设置定时任务

### 使用 Windows 任务计划程序

1. **打开任务计划程序**
   - 按 `Win + R`
   - 输入 `taskschd.msc`
   - 回车

2. **创建基本任务**
   - 右键点击"任务计划程序库"
   - 选择"创建基本任务"

3. **配置任务**
   - **名称**：个人任务提醒
   - **描述**：每天早上提醒我的任务
   - **触发器**：每天
   - **时间**：上午 9:00
   - **操作**：启动程序
   - **程序/脚本**：`C:\Users\Administrator\MyTools\TaskReminder\run_reminder.bat`

4. **完成设置**
   - 勾选"完成时打开此任务的属性对话框"
   - 点击"完成"

5. **高级设置**（可选）
   - 勾选"不管用户是否登录都要运行"
   - 勾选"使用最高权限运行"

### 任务格式

```json
{
  "tasks": [
    {
      "name": "角色动画制作",
      "deadline": "2026-01-25",
      "status": "进行中",
      "priority": "高"
    },
    {
      "name": "UI 界面设计",
      "deadline": "2026-01-22",
      "status": "进行中",
      "priority": "中"
    }
  ],
  "my_email": "wangxinlai@huixuanjiasu.com"
}
```

### 任务状态

- `未开始` - 还没开始做
- `进行中` - 正在做
- `已完成` - 已经完成（不会提醒）

### 优先级

- `高` - 🔴 红色标记
- `中` - 🟡 黄色标记
- `低` - 🟢 绿色标记

## 🔧 常见问题

### Q: 没有收到提醒？

**检查清单**：
1. 任务计划程序是否正常运行？
2. 飞书应用凭证是否正确？
3. 邮箱地址是否正确？
4. 任务是否已标记为"已完成"？

### Q: 如何暂停提醒？

**方法 1**：禁用任务计划
- 打开任务计划程序
- 找到"个人任务提醒"
- 右键 → 禁用

**方法 2**：清空任务列表
- 编辑 `my_tasks.json`
- 将 `tasks` 数组清空：`"tasks": []`

### Q: 如何修改提醒时间？

- 打开任务计划程序
- 找到"个人任务提醒"
- 右键 → 属性
- 触发器 → 编辑
- 修改时间

## 🎯 使用技巧

### 1. 每周回顾

每周日晚上更新下周的任务列表。

### 2. 优先级管理

- 高优先级：紧急且重要
- 中优先级：重要但不紧急
- 低优先级：可以延后

### 3. 任务分解

把大任务拆分成小任务，每个任务 1-3 天完成。
