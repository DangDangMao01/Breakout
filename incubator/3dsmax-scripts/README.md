# 3ds Max 脚本集合

> 存放 3ds Max 相关脚本和工具

## 目录结构

```
3dsmax-scripts/
├── scripts/          # MaxScript 脚本 (.ms, .mcr)
├── plugins/          # 插件文件 (.dlu, .dlm)
└── startup/          # 启动脚本
```

## 使用说明

### 脚本安装
1. 将 `.ms` 文件复制到 `3ds Max 安装目录\scripts`
2. 或通过 `MAXScript > Run Script` 运行

### 插件安装
1. 将插件文件复制到 `3ds Max 安装目录\plugins`
2. 重启 3ds Max
