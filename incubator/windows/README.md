# Windows 工具集

这个文件夹包含用于 Windows 系统的实用工具。

## 工具列表

### SETUNA.exe
**截图置顶工具**

轻量级的屏幕截图置顶工具，可以将截图固定显示在桌面最上层，方便工作时参考。

- [完整使用指南](./SETUNA使用指南.md)
- 适用场景：设计参考、代码对照、文档查阅
- 资源占用：极低（< 10MB）

**快速开始**：
1. 双击 `SETUNA.exe` 启动
2. 点击托盘图标开始截图
3. 拖动选择区域
4. 截图自动置顶显示

## 使用建议

### 与项目工作流集成

这些工具特别适合以下工作场景：

1. **Blender 建模/动画**
   - 截取参考图置顶
   - 对照原画进行建模

2. **ComfyUI 工作流调试**
   - 截取正确的节点配置
   - 对照配置进行调试

3. **Spine 动画制作**
   - 截取关键帧参考
   - 对照序列帧进行调整

4. **脚本开发**
   - 截取 API 文档
   - 边看文档边编码
