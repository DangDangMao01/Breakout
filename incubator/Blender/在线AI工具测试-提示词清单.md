## 📊 提示词优化技巧

### 核心要素（必须包含）

1. **动作描述**：具体动作 + "in place"（原地）
2. **视角**：side view（侧面视角）
3. **风格**：2D game animation style
4. **循环**：smooth loop（流畅循环）
5. **背景**：green screen background（绿幕背景）⭐

### 可选增强

- **速度控制**：slow motion / fast movement
- **情绪强化**：energetic / gentle / dramatic
- **细节描述**：holding book steady / eyes sparkling

### 负面提示词（Negative Prompt）

如果工具支持负面提示词，可以添加：
```
blurry, low quality, distorted, multiple characters, 
inconsistent character, different hair color, 
different clothing, no book, background objects, 
complex background
```

**中文版**：
```
模糊，低质量，扭曲，多个角色，
角色不一致，不同发色，不同服装，
没有书，背景物体，复杂背景
```

## 🔧 后期处理流程

### 第一步：下载视频
- 保存为MP4格式
- 命名规范：`character_idle.mp4`, `character_run.mp4` 等

### 第二步：提取序列帧

**方法A：使用Cocos工具**
- 使用"视频转序列帧图片生成器"
- 自动抠绿背景
- 输出PNG序列帧

**方法B：使用FFmpeg**
```bash
# 提取帧（24fps）
ffmpeg -i character_run.mp4 -vf fps=24 frames/run/frame_%04d.png

# 去绿背景（使用chromakey）
ffmpeg -i character_run.mp4 -vf "chromakey=0x00FF00:0.1:0.2,fps=24" frames/run_nobg/frame_%04d.png
```

### 第三步：转换为Spine动画
- 使用"Spine序列帧动画生成器"
- 设置帧率：18-24fps
- 添加动画事件（可选）
- 生成Spine文件

## 💡 关键提示

### 绿背景的重要性
✅ **为什么用绿色**：
- 与角色颜色对比度高（蓝发、白衣）
- 标准Chroma Key颜色
- 抠图工具支持最好

✅ **替代颜色**：
- 蓝色背景：如果角色有绿色元素
- 纯白/纯黑：简单场景可用

### 角色一致性技巧
✅ **提示词中强调**：
- "same character"（同一角色）
- "consistent appearance"（外观一致）
- "blue hair, white sweater, brown book"（具体特征）

✅ **使用角色参考功能**（如果工具支持）：
- Pika: Character Reference
- Runway: Image Reference
- Kling: Elements功能

### 时长选择
✅ **2秒**：快速动作（跑步、跳跃、挥手）
✅ **3秒**：慢动作（待机、阅读、表情）
✅ **5秒**：复杂动作（如果工具支持）
