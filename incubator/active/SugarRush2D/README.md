## 📖 项目概述

Sugar Rush 2D 是一个糖果主题的 2D 平台跑酷游戏。玩家控制可爱的角色在糖果世界中奔跑、跳跃、收集糖果，躲避障碍物，挑战高分。

### 一句话介绍
在甜蜜的糖果世界中奔跑，收集糖果，挑战极限！

---

## 🎮 核心玩法

### 玩家做什么？
- **奔跑**: 角色自动向前跑
- **跳跃**: 点击屏幕跳跃，躲避障碍
- **收集**: 收集糖果获得分数
- **解锁**: 解锁新角色和道具

### 游戏目标
- 跑得越远越好
- 收集越多糖果越好
- 解锁所有角色

## 🎯 差异化策略

### 与竞品的区别

| 方面 | Temple Run | Subway Surfers | Sugar Rush 2D |
|------|-----------|---------------|---------------|
| 视角 | 3D 第三人称 | 3D 第三人称 | **2D 侧视图** |
| 美术 | 写实 | 卡通 | **糖果主题** |
| 操作 | 滑动 | 滑动 | **点击跳跃** |
| 特色 | 追逐感 | 城市场景 | **糖果世界** |

### 我们的优势
- ✅ 2D 开发成本低
- ✅ 糖果主题有吸引力
- ✅ 操作简单（单点击）
- ✅ 适合短视频传播
