# 孵化器

> 小项目/脚本/插件的孵化地

## 项目状态

| 项目 | 状态 | 说明 |
|------|------|------|
| kiro-knowledge-base |  成熟 | Kiro 知识库插件 v2.2.0 |
| SecondMind-Product |  idea | 产品想法/规划 |

## 状态说明

-  idea - 想法阶段
-  exploring - 探索中
-  developing - 开发中
-  mature - 成熟，可独立
-  graduated - 已独立

## 新建项目

1. 在 `experiments/` 下创建文件夹
2. 添加 `STATUS.md` 标记状态
3. 成熟后参考 `GRADUATION-CRITERIA.md` 决定是否独立
