## 快速开始

### 方式 1：快速聊天（推荐）⭐

**最简单的方式，支持搜索**

双击运行：`快速聊天.bat`

```
AI> Blender 4.0 有什么新功能？
🔍 搜索: Blender 4.0 有什么新功能？
💭 生成回答...

Blender 4.0的主要新功能是...
（带搜索结果和来源）
```

### 方式 2：增强 CLI（完整功能）⭐⭐

**支持搜索 + 记忆系统 + 技术资讯**

双击运行：`启动增强CLI.bat`

### 方式 3：直接对话（无搜索）

双击运行：`简单测试对话.bat`

这会启动 Ollama 的原生对话界面。

## 常见问题

### Q1: 搜索失败怎么办？

**原因：** 网络问题或搜索 API 限制

**解决：** 系统会自动降级到模型原有知识

### Q2: 如何强制搜索？

**方法 1：** 在问题中加入关键词（最新、2026等）

**方法 2：** 使用增强 CLI 的 `search` 命令

### Q3: 如何获取技术资讯？

使用增强 CLI：

```
AI> news AI       # AI 领域
AI> news Blender  # Blender 领域
AI> news Unity    # Unity 领域
```

### Q4: 依赖安装失败怎么办？

使用简化版：

- `快速聊天.bat` - 只需要 ddgs 和 requests
- `简单测试对话.bat` - 不需要任何依赖
